### 外卖优惠券cps,美团饿了么外卖红包，淘宝客，京东客

### 介绍
外卖红包CPS前端开源项目，兼容淘宝客，无需用户淘宝授权即可跟单，已接入淘宝联盟、美团联盟、美团分销，京东、拼多多已经规划中。

### 软件架构
线上正在使用项目，基于uniapp跨端架构，UI框架基于uview开发。

### 使用方法
源码为uniapp项目，需下载hbuilder导入项目打包，可编译成h5或小程序和app
